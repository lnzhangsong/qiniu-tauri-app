# qiniu tauri app

## 相关指令

1. 初始化

    ```shell
    pnpm init
    pnpm tauri ios init
    pnpm tauri android init
    ```

2. 开发

    ```shell
    pnpm run dev
    pnpm tauri build
    pnpm tauri dev
    pnpm tauri ios dev
    pnpm tauri android dev
    ```
